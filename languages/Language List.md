**Basics**

| Language                 | Why Learn It Now                                                                                 |
| ------------------------ | ------------------------------------------------------------------------------------------------ |
| **Markdown (MD)**        | You'll use this everywhere — README, notes, docs. Learn it in 1 hour.                            |
| **Lua**                  | Lightweight, fast, used in Neovim config, game scripting. Very clean syntax.                     |
| **Bash/Shell scripting** | You're already in Linux — automate everything. `if`, `for`, functions, scripts. Essential.       |
| **TOML**                 | Config file format used in Neovim, Rust, many tools. 30 mins to learn.                           |
| **JSON**                 | Universal data format. You'll see it everywhere. Learn structure + `jq` tool.                    |
| **AWK**                  | Already in Linux commands but deserves its own time — tiny but powerful text processing language |
